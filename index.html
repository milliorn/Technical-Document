Hello World Code Snippets

.NET
Console.WriteLine Method
Writes the specified data, followed by the current line terminator, to the standard output stream.
        
using System;

namespace myApp
{
  class Program
  {
    static void Main(string[] args)
    {
      Console.WriteLine("Hello World!");
    }
  }
}
Java
System.out.println()
In Java, System.out.println() is a statement which prints the argument passed to it. The println() method display results on the monitor. Usually, a method is invoked by objectname.methodname().
    
public class HelloWorld {

  public static void main(String[] args) {
    // Prints "Hello, World" to the terminal window.
    System.out.println("Hello, World");
  }

}
JavaScript
console.log()
The console.log() method outputs a message to the web console. The message may be a single string (with optional substitution values), or it may be any one or more javaScript objects.
    
console.log('Hello World');
TypeScript
console.log()
The console.log() method outputs a message to the web console. See JavaScript above.
    
let message: string = 'Hello, World!';
console.log(message);
Python
print()
Print objects to the text stream file, separated by sep and followed by end. sep, end, file, and flush, if present, must be given as keyword arguments.
    
print("Hello, World!")
